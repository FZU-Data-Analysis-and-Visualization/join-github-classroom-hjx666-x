### Python数据分析代码
```python
import pandas as pd
# 读取销售数据
data = pd.read_csv("sales_data.csv")
# 计算销售额均值
avg_sales = data["销售额"].mean()
print(f"平均销售额：{avg_sales:.2f}万元")
代码功能：读取销售数据文件并计算月度销售额的平均值，快速获取核心统计指标。
 
数据分析常用公式
 
均值公式：
$\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$
公式说明：用于计算样本数据的平均值，是描述性统计的核心指标之一。
 
方差公式：
$\sigma^2 = \frac{1}{n}\sum_{i=1}^{n} (x_i - \bar{x})^2$
公式说明：用于衡量数据的离散程度，反映数据与均值的偏离情况。
